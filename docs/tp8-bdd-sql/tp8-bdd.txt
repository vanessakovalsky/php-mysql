- Avec l'aide de phpMyAdmin, ou du script SQL bdd.sql créer $
une base de donnée appelée kingoludo encodée en utf8_general_ci
- Créer une table jeux, qui reprend les 9 champs du CdC + 
un champ id en autoincrement
- Créer une table user, qui reprend les 8 champs du CdC
- Créer une table user_jeux qui contient deux colonnes :
-- id_user
-- id_jeux 
- Chacune de ces colonnes sera une référence vers un id dans la table concernée
- A partir des fichiers jeux.txt et user.txt insérer des données en base de données
- Générer un export sql de votre structure de table + des données 
- Créer un fichier connect.php qui contient les informations de connexion
à votre base de données
- Dans le fichier afficher_jeu.php récupérer les données du formulaires et les insérer
dans la table jeu à l'aide de PDO 
- Dans le fichier liste-jeux.php afficher les données des jeux en provenance de la base 
de données et non plus du fichier
- Dans le fichier connexion.php vérifier l'identifiant et le mot de passe à partir
des informations contenus en base de données et non plus dans le fichier txt
